## Architecture Design

- [Power 3-4 Architecture Design (English)](./architecture/power-3-4-design-EN.md)
- [Power 3-4 架构设计 (中文)](./architecture/power-3-4-design-CN.md)

This design describes the architecture for Powers 3-9, which handle complete Cloudflare CDN configuration migration to AWS CloudFront.

**Key Features:**
- Domain-based rule grouping (meets CloudFront Function 10KB limit)
- Implementation-based task assignment
- Multi-stage conversion workflow
- Cost-aware design
- Stateless workflow
